  • Bug Fix: Remember the selected Destination Playlist between searches on the Search Tab.

  • Tweak: Make finding a very close match case-insensitive.

  • Tweak: When mouse is over a button change the cursor to a pointer.

2022/08/20

  • Tweak: Improved the help text.

2022/07/23

  • Bug Fix: Handle race condition bewteen spotify removing tracks and this app refetching the playlist.

2022/06/30

  • Tweak: Rearranged tabs so that a user is more likely to try the search feature.

2022/04/28

  • Bug Fix: When removing 100 tracks at a time, (by pos or by id), ignore tracks with a null track id.

2022/04/03

  • New Feature: Added support for handling podcast episodes.
  • Tweak: Auto Select on Dups Tab UI now has its own box.
  • Tweak: Remove tracks by position errors now include playlist name and track name.

2022/03/26

  • Tweak: Errors now include line numbers.

2022/02/05

  • Tweak: Added hint on Search tab to enter value into Search Text box.

2022/01/06

  • Bug Fix: If a track has a null track id do not allow the track to be removed, copied or moved to prevent an error from occurring.
  • Bug Fix: Duration matching fixed when finding dups across playlists when the first track is shorter than the second track.
  • Bug Fix: The defualt duration was changed from 30 to 10 seconds and now can be changed by the user.

2021/12/10

  • New Feature: If the user selects playlists with more than 30,000 tracks a dialog is displayed recommending the user select fewer playlists.
  • New Feature: If a Remove (track by position) fails the error dialog now explains why this happens and what the user needs to do.
  • New Feature: The user's country code is now fetched at login to determine which language translations are needed.
  • Bug Fix: The search tab results are cleared when all the characters in search text input are removed by the backspace or delete keys.

2021/12/03

  • Bug Fix: Switched from buymeacoffee/spotifinder to ko-fi/spotifyfinder. paypal no longer available on buymeacoffee.

2021/11/19

  • New Feature: The Duplicates Tab now has a Remove Dups by Playlist feature. See Duplicates Tab help page for details.
  • Bug Fix: The Duplicates Tab Auto Select now selects up to 100 tracks at a time (removing 100 tracks/call is a Spotify limit).
  • Bug Fix: On any tab: when a user tries to select more than 100 tracks an alert dialog is displayed (removing 100 tracks/call is a Spotify limit).

2021/11/01

  • Bug Fix: Session timeout issued fixed (sessions timeout after 12 hours of no activity)

2021/10/27

  • New Feature: The new Search Tab provides an easy way to search your playlists.

2021/08/28

  • New Feature: On the Playlist Tab you can now Save your playlist selections. They will be automatically selected on your next visit.

2021/05/08

  • SpotifyFinder.com goes live.
