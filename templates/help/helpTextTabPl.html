Playlist Tab

Overview

  • the playlists selected on this tab determime what playlists are examined in the tracks, duplicates, and artist tabs
  • displays all the playlists you have added to your Spotify account
  • by default all the playlists that have your User Id are selected
  • by default all the playlists that have your User Id are displayed at the top of the table

Select Playlist by User Id (drop down combo box)

  • selects playlists based on User Id
  • playlists you have created will have your User Id
  • playlists that you "Saved to Your Library" will have the User Id of user that created the playlist
  • playlists that you "Saved to Your Library" created by Spotify will have a User Id = Spotify

Select All (buttton)

  • every playlist will be selected regardless of the user id

Clear All (button)

  • every playlist will be deselected regardless of the user id

Reload Playlists from Spotify (button)

  • reloads all of your playlists you created or saved to you library on Spotify
  • reloading your playlists can be useful if you change any of your playlists in the Spotify App
  • reloading your playlists can be useful if you changed the column sort and want to revert to the original sorting

Finding a Playlist

  • click in the search box at the bottom of PlaylistName column and then type the first few letters of the playlist name

Sorting (column headers)

  • you can click on any column header to sort the table
  • to return to the original sort order press the refresh button

Refresh (Button)

  • will return the table to the original sort order
  • if you changed a playlist in the Spotify App press 'Reload Playlists from Spotify'
