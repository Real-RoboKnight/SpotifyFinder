Artist Tab

Overview

  • displays all the tracks for each artist found in your selected playlists
    • add/remove selected playlists on the Playlist Tab
  • remove a track from specific playlist

Finding an Artist

  • click in the Artist Name column and then type the first few letters of the artist name

Remove Selected Tracks

  • tracks with a checkmark present will be removed from the associated playlist

Reload Artist Track Table

  • reloading your artist track table can be useful if you changed the column sort and want to revert to the original sorting
  • if you changed a playlist in the Spotify App go to the Playlist Tab and press 'Reload Playlists from Spotify

Space Bar Select/Deselect

  • When a row is highlighted pressing the space bar will select or deselect the row
  • Use the up/down arrows and space bar to select and deselect a row

Sorting (column headers)

  • you can click on any column header to sort the table
  • to return to the original sort order reload the playlists
