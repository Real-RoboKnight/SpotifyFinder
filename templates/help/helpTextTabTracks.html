Tracks Tab

Overview

  • display track(s) for a specific playlist
  • the playlist table reflects the playlists selected on the Playlist Tab
  • remove a track from a specific playlist

Remove Tracks from Playlist (button)

  • tracks with a checkmark present will be removed from the associated playlist

Reload Track Table (button)

  • reloading the track table can be useful if you changed the column sort and want to revert to the original sorting
  • if you changed a playlist in the Spotify App go to the Playlist Tab and press 'Reload Playlists from Spotify'

Finding a Playlist

  • click in the Play List Name column and then type the first few letters of the playlist name

Finding a Track

  • click in the Track Name column and then type the first few letters of the track name

Space Bar Select/Deselect

  • When a row is highlighted pressing the space bar will select or deselect the row
  • Use the up/down arrows and space bar to select and deselect a row

Sorting (column headers)

  • you can click on any column header to sort the table
  • to return to the original sort order press reload track table

Number of Tracks do not match Spotify's Number of Tracks

  • this indicates that the playlist contains unplayable track(s) (the track(s) were once playable but no longer are)
  • this can occur when the artist has removed a track or album from Spotify
  • this can occur if the track is no longer available in your country
  • you can reach out to the artist and let them know something has changed and they may fix the issue

Making a playlist track order match Spotify's track order

  • In this app sort on the Duration column and in the Spotify app sort on duration
