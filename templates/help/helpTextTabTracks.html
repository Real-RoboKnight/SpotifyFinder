Tracks Tab

Overview

  • display tracks for a specific playlist
  • the list of playlists shown reflects the playlists selected on the Playlist Tab
  • remove a track from a specific playlist
  • move a track to a specific playlist
  • copy a track to a specific playlist

Remove Selected Tracks (button)

  • tracks with a checkmark present will be removed from the associated playlist
  • you can not remove tracks from a playlist you do not own

Move Selected Tracks (button)

  • tracks with a checkmark present will be moved to the destination playlist
  • you can not move tracks from a playlist you do not own
  • the destination playlist drop down box shows playlists selected on the Playlist Tab that you own
  • you can only move tracks to playlists you own

Copy Selected Tracks (button)

  • tracks with a checkmark present will be copied to the destination playlist
  • you can copy tracks from a playlist you do not own
  • the destination playlist drop down box shows playlists selected on the Playlist Tab that you own
  • you can only copy tracks to playlists you own

Finding a Playlist

  • click in the search box at the bottom of PlaylistName column and then type the first few letters of the playlist name

Finding a Track

  • click in the search box at the bottom of TrackName column and then type the first few letters of the track name

Sorting (column headers)

  • you can click on any column header to sort the table
  • to return to the original sort order press the refresh button at the bottom of the page

Create Playlist (button)

  • creates a new playlist from the selected tracks.
  • to create an ordered/sorted playlist click on a column header prior to pressing Create Playlist.
  • for example to create a playlist sorted by artist/album:
    1. goto Tacks Tab
    2. select a playlist in the PlaylistName table
    3. click on the AlbumName column header
    4. click on the ArtistName column header
    5. click on Select All
    6. edit playlist name
    7. click on Create Playlist

Save Sort (button)

  • used to change the sort an existing playlist.
  • a new playlist is created with a new playlist id using the existing playlist name.
  • the new playlist will have all the tracks of the existing playlist.
  • the existing playlist is deleted.
  • do not do this if the existing playlist has followers or is shared with others
  • for example to change the sort of an existing playlist:
    1. goto Tacks Tab
    2. select a playlist in the PlaylistName table
    3. click on the AlbumName column header
    4. click on the ArtistName column header
    5. click on Save Sort

Select All (button)

  • selects all the tracks in the tracks table

Clear (button)

  • clears all selections and returns the table to the original sort order
  • refreshing your track table can be useful if you changed the column sort and want to revert to the original sorting
  • if you changed a playlist in the Spotify App: press 'Reload from Spotify' button to pull in the changes

Reload from Spotify (button)

  • reloads all of your playlists from Spotify
  • reloading your playlists can be useful if you change any of your playlists in the Spotify App while using this website

Grayed Out Tracks in Spotify Playlists

  • greyed out indicated that the track(s) were once playable but no longer are
  • this can occur when the artist has removed a track or album from Spotify
  • this can occur if the track is no longer available in your country
  • you can reach out to the artist and let them know something has changed and they may fix the issue

Null Track Id's

  • tracks with null track ids can not be removed
  • null track ids can occur if spotify lost the rights to the track or artist

Making a playlist track order match Spotify's track order

  • In this app sort on the Duration column and in the Spotify app sort on duration
