Duplicates Tab

Overview

  • displays all duplicates tracks for the selected playlists
    • add/remove selected playlists on the Playlist Tab
  • remove a duplicate track from a specific playlist

Finding Dups (radio buttons)

  • there are 4 ways to search for duplicates:
    1. find dups across playlists using track id (exact match)
    2. find dups across playlists using "track name/artist name/track duration" (a very close match)
    3. find dups in the same playlist using track id (exact match)
    4. find dups in the same playlist using "track name/artist name/track duration" (a very close match)

When finding dups using "track name/artist name/track duration"

  • the track name must be identical
  • the artist name must be identical
  • the track duration with 30 seconds

Why do tracks have the same name but different track ids

  • There are many reasons:
    1. The artist has made multiple recordings of the same track
    2. The artist releases the same track on multiple albums, original, greatist hits
    3. The artist re-releases the same album with improved sound (remastered)
    4. It is a long list....

Remove Selected Duplicates (button)

  • tracks with a checkmark present will be removed from the associated playlist

Reload Dup Table (button)

  • reloading your dups table can be useful if you changed the column sort and want to revert to the original sorting
  • if you changed a playlist in the Spotify App go to the Playlist Tab and press 'Reload Playlists from Spotify'

Space Bar Select/Deselect

  • When a row is highlighted pressing the space bar will select or deselect the row
  • Use the up/down arrows and space bar to select and deselect a row

Sorting (column headers)

  • you can click on any column header to sort the table
  • to return to the original sort order press reload dups table
